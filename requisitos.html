Requisitos para vuelos nacionales e internacionales

Requisitos para vuelos nacionales

Para los vuelos nacionales, asegúrate de cumplir con los siguientes requisitos:

  • Documento de identificación válido.
  • Comprobante de reserva del vuelo.
  • Revisar las restricciones de equipaje de la aerolínea.
  • Realizar el check-in con anticipación.

Requisitos para vuelos internacionales

Para los vuelos internacionales, debes cumplir con los siguientes requisitos adicionales:

  • Pasaporte válido.
  • Visa (si es necesaria para tu destino).
  • Certificado de vacunación (si es requerido).
  • Declaración de aduana.
